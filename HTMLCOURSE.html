Let's Start!

OVERVIEW

HTML or Hypertext Markup Language is a standardized system of tagging text files to achieve font, color, graphic, and hyperlink effects on World Wide Web pages. HTML is the backbone of any web pages, it instructs the browser on how to display datas on the browser. HTML is composed of different tags and tags has different attributes. You will know more as we dive into our lesson.

HTML TAGS

HTML tags are the elements used to instruct the browsers on how to display datas.

SOME HTML TAGS USED IN DISPLAYING A SIMPLE WEB PAGE

<!DOCTYPE html> : Used to declare the html version.

<html> </html> : Used to enclose the whole html code.

<head> </head> : Used to enclose the element that are not visible.

<title> </title> : Used to display the title of the web page at the top of the browser.

<body> </body> : Used to enclose the elements for the data that are visible in the viewport.

<p> </p> : Used to enclose a paragraph.

<img> : Used to display image.

Making your first web site.

Before we begin, we need a text editor, it's where we write our code. There are alot of text editors out there like Atom, VS Code, but in this tutorial we will use Sublime Text editor and we'll use the latest version of HTMl which is the version5.

Steps

First, type: <!DOCTYPE html>

Then type: <html>

Then type: <head> </head>

Then type: <title> </title>

Then type: <body>

Then type: <p> </p>

Then type: </body>

Then type: </html>

The code in your text editor should look like this.


Now lets display some text.

Inside the <title> </title> type any title you want, then run the browser and notice that there is nothing displayed, it's because titles are displayed in the upper part of your browser./p>

Inside the <p> </p> tag, write your name or anything. After that run the html file and you will see the text is displayed in the browser.
